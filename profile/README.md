# Codificando Ideias

**Impulsionando negócios através da tecnologia.**

Desenvolvemos sites, sistemas, ferramentas internas e plataformas digitais sob medida para empresas que querem crescer com estrutura, clareza e presença online profissional.

---

## Visão Geral

A **Codificando Ideias** é organizada como um ecossistema digital modular.

Cada repositório tem uma responsabilidade específica dentro da jornada do usuário — desde a descoberta da marca, passando pela conversão em lead, até a gestão comercial e a entrega de projetos para clientes.

### Estrutura simplificada

```text
Internet
   │
   ▼
Landing Page
   │
   ▼
Supabase Database
   │
   ├── Admin (CRM)
   ├── Portal do Cliente
   └── Newsletter
``` 

Além disso, o ecossistema inclui projetos voltados para conteúdo técnico, SEO e demonstração de capacidade técnica.

## Repositórios Principais

### Landing Page
Site institucional da marca.

### Responsabilidades
```
- Apresentação da marca

- Proposta de valor

- Conversão de visitantes em leads

- Captura de newsletter
```
### Tecnologias
```
- HTML5

- CSS3

- Bootstrap

- JavaScript

- Supabase

- GitHub Pages
```
## Templates e Exemplos
Repositório com exemplos de projetos e templates reutilizáveis.

### Objetivos
```
- Demonstrar capacidade técnica

- Servir como catálogo de soluções

- Disponibilizar bases reutilizáveis
```
## Admin
Painel administrativo da operação comercial.

### Funções
```
- Gestão de leads

- Pipeline comercial

- Geração de propostas

- Gestão de contratos

- Controle do funil de vendas
```
## Portal do Cliente
Ambiente utilizado pelos clientes após o fechamento do contrato.

### Funções
```
- Visualização de propostas

- Acompanhamento de status

- Links de homologação e produção

- Gestão de documentos
```
## Blog
Plataforma de conteúdo técnico da marca.

### Objetivos
```
- Publicação de artigos

- SEO técnico

- Construção de autoridade

- Geração de tráfego orgânico
```
## Newsletter
Sistema de distribuição de conteúdo e relacionamento com a comunidade.

### Funções
```
- Gestão de inscritos

- Envio de campanhas

- Comunicação com a comunidade
```

# Backend e Dados
